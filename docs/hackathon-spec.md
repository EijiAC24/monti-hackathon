## 1. プロジェクト概要

### 1.1 エレベーターピッチ

**「親が言いたいことを、子供が大好きなAIキャラが代わりに伝える」**

50年以上の心理学研究に基づいた"科学的しつけアプリ"。
恐怖や命令ではなく、モンテッソーリ教育とソクラテス式の質問で子供が自分で気づき、自分で行動できるよう導く。

### 1.2 ハッカソンカテゴリ

**Live Agents 🗣️** — リアルタイム音声対話エージェント

### 1.3 なぜLive Agents？

- Gemini Live APIによるリアルタイム双方向音声会話がコア機能
- 子供の割り込み（barge-in）に自然に対応
- VAD（Voice Activity Detection）で子供はボタン操作不要
- 音声のトーンに応じた感情対応（Affective Dialog）

### 1.4 差別化（他の参加者がやらないこと）

| 一般的なLive Agent作品 | Monti |
|----------------------|-------|
| 汎用チャットボット | 子供向け特化UX |
| テキスト + 音声 | 音声オンリー（子供が操作しやすい） |
| 技術デモ止まり | 心理学的根拠に基づく設計 |
| 既存カテゴリ（翻訳、サポート等） | 「しつけ × AI」という新カテゴリ |

## 2. 理論的基盤（3本柱）

> 審査基準 Innovation & Multimodal UX（40%）で高得点を狙うための核

### 2.1 モンテッソーリ教育

自己教育・自由選択・教師はガイド役。答えを教えず、質問で子供が自分で気づけるよう導く。

### 2.2 ソクラテス式メソッド

質問を通じて相手が自ら真理に到達するよう導く対話法。5-6歳児への研究で批判的思考スキルの有意な発達が確認されている。

### 2.3 自己決定理論（SDT）

Ryan & Deci (2000)。自律性・有能感・関係性の3つの基本欲求を満たすことで内発的動機づけが高まる。

### 2.4 従来のアプローチとの対比（デモ動画で使う）

| 観点 | 従来のしつけ | Monti |
|------|------------|-------|
| アプローチ | 命令・脅し（"Do it or else!"） | ポジティブ強化で気づかせる |
| 心理的効果 | リアクタンス（反発） | 内発的動機づけ |
| 会話 | 一方的な指示 | リアルタイムAI双方向対話 |
| 教育理論 | なし（経験則） | モンテッソーリ + ソクラテス + SDT |
| 持続性 | 低い（親がいないとやらない） | 高い（自分で気づいたから続く） |

### 5.4 会話画面（P0 — 最重要）

**操作方式**: VAD（Voice Activity Detection）
- ボタン操作不要、子供は普通に話すだけ
- キャラが話し終わると自動的に「聞くモード」
- 子供が話し始めたらキャラは即停止（barge-in）

```
┌─────────────────────────┐
│ ⏸️                  ✕    │  ← 一時停止 / 終了
├─────────────────────────┤
│                         │
│                         │
│        🐻               │  ← 画面上部 60%
│      (アニメーション)     │     状態に応じて表情変化
│                         │
│  ┌───────────────────┐  │
│  │「じゃあどうしたら  │  │  ← 吹き出し（テキスト表示）
│  │  いいとおもう？」  │  │
│  └───────────────────┘  │
│                         │
├─────────────────────────┤
│                         │
│     🎧 きいてるよ...    │  ← 状態インジケーター
│     ○ ○ ○ (波形)       │
│                         │
└─────────────────────────┘
```

**キャラクター状態:**

| 状態 | トリガー | 表示 |
|------|---------|------|
| idle | デフォルト | ゆらゆら揺れ、「はなしてね！」 |
| listening | 子供が話し始め | うなずき、「きいてるよ...🎧」 |
| thinking | 応答生成中 | 首かしげ、「うーんと...」 |
| talking | キャラ発話中 | 口パク、吹き出しにテキスト |
| happy | ゴール達成時 | 飛び跳ね、「やったー！✨」 |

**アニメーション実装（ハッカソン版）:**
- Riveは使わない（作成時間がない）
- Flutter AnimationController + カスタムウィジェット
- キャラは静止画（PNG/SVG）+ transformアニメーション
- 最低限: idle揺れ + talking口パク + happy跳ね

## 6. システムプロンプト設計

### 6.1 ベースプロンプト

```
あなたは「モンティ」という名前の、優しいクマのAIキャラクターです。
{child_name}ちゃん（{age}歳）と楽しくおしゃべりします。

【重要な原則】
1. 命令しない。質問で導く（ソクラテス式）
2. 答えを教えない。子供が自分で気づけるように
3. 努力やプロセスを褒める（「頑張ったね」「自分で気づけたね」）
4. 能力や人格を評価しない（「えらいね」「いい子だね」は避ける）
5. 恐怖や脅しは絶対に使わない
6. 子供の名前を呼んで親しみを持たせる
7. {interests}の話題を織り交ぜる

【今日のシナリオ】
テーマ: {scenario_title}
ゴール: {scenario_goal}

【会話の流れ】
1. 挨拶と雑談で関係構築（30秒）
2. シナリオに関連する質問を投げかける
3. 子供の答えを受けて、さらに深掘り質問
4. 子供が自分で答えにたどり着いたら承認
5. 行動を促す（命令ではなく提案として）
6. 完了したら達成を一緒に喜ぶ

【話し方のルール】
- 短い文で話す（1文15文字以内目安）
- 1回の発話は2-3文まで
- 子供が知っている言葉だけ使う
- 「〜だよね」「〜かな？」など柔らかい語尾
- 子供の発言をオウム返しで受け止めてから応答

【禁止事項】
- 「〜しなさい」「〜しないとダメ」などの命令形
- 「鬼が来るよ」「お化けが来るよ」などの脅し
- 「えらいね」「いい子だね」などの人格評価
- 長すぎる説明
- 難しい言葉
- 親や大人の話題
```

> Devpost提出用

**Monti: Your Child's AI Companion for Positive Habit Building**

Monti is an AI-powered voice companion that helps children aged 2-8 develop healthy habits through natural conversation, not fear or commands.

Built on three pillars of developmental psychology (Montessori education, Socratic method, and Self-Determination Theory), Monti uses the Gemini Live API for real-time voice interaction where children can speak naturally, interrupt freely, and receive emotionally responsive feedback.

Every parent knows the struggle: "Brush your teeth!" leads to resistance, not action. This is Psychological Reactance, a well-documented phenomenon where children push back against direct commands. Monti solves this by replacing commands with guided questions, helping children discover the right actions on their own and building intrinsic motivation that lasts.

**Key Features:**
- Real-time voice conversation with an AI bear character "Monty"
- Socratic questioning that guides children to their own conclusions
- Personalized interactions using the child's name and interests
- Parent dashboard with AI-generated conversation insights
- Built with Gemini Live API, Cloud Run, and Firestore on Google Cloud

**Category:** Live Agents
